# QR Code Generator

A lightweight web app that instantly generates QR codes from any text or URL.

## Features
- Enter any text or URL to generate a QR code
- Smooth reveal animation when the QR code appears
- Input validation with shake animation on empty submission
- Clean, minimal UI with a centered card layout

## Tech Stack
- HTML
- CSS
- JavaScript
- [QR Server API](https://goqr.me/api/) for QR code generation

## Usage
1. Open `index.html` in your browser
2. Type any text or paste a URL
3. Click **Generate** to see your QR code
